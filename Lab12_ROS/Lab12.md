**ROS** (Robot Operating System) 

---


### 1. 主題

本次實驗的主題是 **機器人作業系統（ROS）** 。

### 2. 實驗過程

* 
**環境建置**：在 Linux 系統上安裝 ROS 2 版本（如 Humble 或 Foxy） 。


* 
**節點操作**：學習如何啟動並運行 ROS 2 節點（Nodes） 。


* 
**通訊實作**：使用 `turtlesim` 模擬器實作發布者（Publisher）與訂閱者（Subscriber）之間的通訊 。


* 
**工具檢測**：利用指令工具觀察 Topic 列表與節點間的數據傳輸 。



### 3. 學習成果

* 掌握了 ROS 2 的基本架構與分散式運算的概念 。


* 學會透過終端機指令控制模擬機器人，並理解節點通訊的基礎邏輯 。

### 1. Topic

The theme of this lab is **Robotic Operating Systems (ROS)**.

### 2. What I Did

* 
**Environment Setup**: Installed ROS 2 distributions (such as Humble or Foxy) on a Linux system.


* 
**Node Operations**: Learned how to initialize and run ROS 2 nodes.


*

* 
**Tool Inspection**: Used command-line tools to monitor topic lists and data transmission between nodes.



### 3. What I Learned

* Mastered the basic architecture of ROS 2 and the concept of distributed computing.


* Learned to control simulated robots via terminal commands and understood the fundamental logic of node communication.
